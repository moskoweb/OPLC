Metabox para auxiliar na organização das páginas na lista com cores.

== Description ==

Plugin cria uma metabox nas páginas, permitindo inserir um texto de descrição e também as cores que deseja modificar a linha da listagem das páginas.

Podendo assim identificar cada tipo de páginas com uma cor diferente na listagem.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/oplc` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Edit page or add new page, with metabox.
